Heading/Action buttons
"Music Wars" heading turns green and attack/defend buttons appear when players encounter

Reset the game
Clicking on the "Music Wars" heading will reset the game

Game Elements

Weapons
 Quarter note10% damage
Eighth note  5% damage
Half note20% damage
Whole note40% damage

Progress Bar
Bar changes color based on health percentage
Health value changes based on damage
Progress changes with health percentage

When the block containing a weapon is passed over or landed on,
the current weapon is placed in the new weapon's place.

When the block containing a player is clicked,
the possible moves are shown.

Game Rules
Players take turns moving. If one encounters the other, they have the option to attack or defend,
If defend is chosen instead of attack, only 50% of the weapon's damage occurs.

Think of it like music...Notes have value...Watch out for the whole rests!
